Resume Builder - Web App

Project Overview

Resume Builder is a multipage web application designed to help users create professional resumes with real-time previews. Built using HTML, CSS, JavaScript, jQuery, PHP, and MySQL, it includes user authentication, data storage, form validation, accessibility features, and a modern user experience.

Implemented Features

  • Frontend: Responsive layout using HTML5 and CSS3
  • JavaScript/jQuery: Live resume preview and client-side interactivity
  • Form Validation: Input validation for required fields and formats
  • Authentication: Secure login and registration with PHP/MySQL
  • Resume Form: Collects name, phone, education, experience, skills, photo
  • Live Preview: Updates resume instantly as user types
  • Database Integration: Stores user and resume data in MySQL
  • Session Management: Protected pages and logout functionality
