Memory King - Mobile App

Project Overview

Memory King is a Flutter-based mobile game that challenges players to match cards quickly and accurately. The app features Firebase Authentication, Firestore integration for real-time score tracking, profile customization, and analytics. It reflects a full-stack mobile app with clean UI and responsive performance.

Implemented Features

  • Firebase Integration: App is fully linked with Firebase project
  • Login Screen: Includes form validation and error handling
  • Sign-Up Screen: Email & password validation with error alerts
  • CRUD Operations: Game scores saved and updated via Firestore
  • Functionality Demo: Complete and functional app demo
  • Firebase Code Understanding: Explained during evaluation
  • User Profile Management: Editable profile with stats and avatar
  • Media Upload & Display: Supports image upload for profile
  • Firebase Analytics: Monitors user activity within the app
